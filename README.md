# Leapot Backend Task

This is a Node.js backend project set up with Express, Mongoose, bcrypt, and JWT for user authentication. The project includes a simple setup for user data management and authentication using MongoDB.

## Features

- User registration with hashed passwords
- User login with JWT-based authentication
- Environment variable configuration using a `.env` file
- MongoDB for data storage

## Getting Started

### Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (version 14 or higher)
- [MongoDB](https://www.mongodb.com/) (local or Atlas)
